# Balık Hesaplama Asistanı (HSA Calculator)

Balıkçılar için geliştirilmiş; Şok Lideri, İp Kalınlığı, Makine ve Kamış uyumunu hesaplayan modern bir web uygulamasıdır.

## Özellikler
*   **HSA (Makinadan Atış) Hesaplayıcısı:** Kaç gram atacağınıza göre ideal ip (PE), şok lideri ve makine boyunu önerir.
*   **Spin (At-Çek) Modülü:** Levrek, Lüfer gibi hedef balıklara göre ekipman önerisi.
*   **Surf (Beklemeli) Modülü:** Yemli ve uzak atış avları için detaylı hesaplama.
*   **Akıllı Bütçe Asistanı:** Bütçenize (Örn: 5000 TL) en uygun Kamış+Makine+İp setini önerir.
*   **PWA Desteği:** Telefonunuza uygulama olarak kurulabilir.
*   **Dil Desteği:** Türkçe ve İngilizce (Otomatik algılama).

## Kurulum ve Çalıştırma

```bash
npm install
npm run dev
```
